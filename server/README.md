## Overview

Server.

## Tech stacks

- Core tech: Typescript/Node.js
- Framework: NestJS
- ORM: Prisma
- Database: MySQL (Changeable)

## Installation

```bash
$ npm install
```

## Migration

```bash
$ npx migrate dev --name init
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
